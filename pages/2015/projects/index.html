Plex Requests

“Simple automated way for users to request new content for Plex.”

I’m currently the developer behind the application Plex Requests. For those who are running Plex servers, it’s a web application that you can run that gives your users easy access to make requests for things to be added to the server. You can find more information about it on GitHub, or visit the project’s website!
